Architecture Features:
  • ZFS snapshots via Sanoid
  • Offsite backups to Backupserver via Syncoid
  • Specialized VMs for services (Pihole, Samba, etc.)
  • VMs cloned from Alma Linux template using Ansible
  • VM configuration automation via Ansible
  • Shared storage for VMs via virtiofs
  • Full LAN access via Tailscale subnet router
  • Secondary offline backup via external drive
  • Redundant DNS and subnet router with secondary Proxmox server
